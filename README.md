# ExchangeApp
A demo exchange app developed using this technologies:
- Kotlin
- MVVM
- Room
- Coroutines
- Retrofit

For rates I used an API from:
```
http://api.exchangeratesapi.io
```
An initial 1000 EUR set when install app then you can convert it to other currencies.
